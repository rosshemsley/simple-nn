# SimpleNet

![build status](https://circleci.com/gh/rosshemsley/simple-nn.png?style=shield)

A simple feed-forward neural network library in C++11. Uses Armadillo for fast  linear algebra.

A project to get better at NNs.

### Requirements

- cmake
- armadillo
- A modern C++ compiler

### Compiling

```
mkdir build
cd build
cmake ..
make
```

### Testing

```
make test
```
